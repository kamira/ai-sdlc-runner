## Security note
The `command` and `api` backends introduce local command execution and outbound HTTP, but only when
the user opts in (config/`--backend`); API keys come from environment variables, never config/repo;
and the executor never bypasses a halt gate. Default `stub` keeps the runner inert/offline.

tested.)
